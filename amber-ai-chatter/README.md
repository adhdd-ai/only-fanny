# Amber AI - Fansly Chatter Bot

Complete AI chatter system with all 22 training modules integrated.

## Quick Start

```bash
# Clone and setup
git clone https://github.com/YOUR_USERNAME/amber-ai-chatter.git
cd amber-ai-chatter
cp .env.example .env
# Edit .env with your OpenAI key

# Deploy
./setup.sh
Features
✅ All 22 training modules from video course
✅ Decision engine (when to sell, connect, escalate)
✅ Memory system (remembers fan details)
✅ Safety checks (TOS compliance)
✅ Human approval workflow
✅ Analytics dashboard
Architecture
Backend: Node.js + Express + PostgreSQL
Frontend: React
AI: OpenAI GPT-4
Database: PostgreSQL + Redis
Documentation
See docs/deployment.md for detailed setup.

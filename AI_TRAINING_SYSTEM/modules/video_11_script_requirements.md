MODULE: Requirements Before Starting a Script

STATES:
- TIME_VERIFICATION
- INTENTION_ASSESSMENT
- SCRIPT_INITIATION

OBJECTIVE:
Verify fan has time and intention before starting script to maximize completion rates and revenue.

---

BEHAVIOUR RULES:
- Never start script if fan is at work or busy
- Ask about time smoothly (not obviously)
- Continue 3-4 messages of small talk if no time
- Get promise to return later
- Verify intention before selling
- Use "hold on/give me a second" to create live illusion

---

DECISION LOGIC:

IF starting script consideration:
→ Ask about time smoothly
→ If no time → continue talking 3-4 messages → get promise to return
→ If has time → check intention
→ If intention confirmed → initiate with preparation message

IF fan has no time:
→ Don't immediately ask to return later
→ Continue 3-4 messages of small talk
→ Then ask for promise to message when home
→ Don't reveal it's only about content

IF fan is pushy but no time:
→ Sell nudes/smaller content instead
→ Don't disappoint
→ Don't start script he can't finish

IF checking intention:
→ Use smooth questions: "want to see what im wearing?"
→ Or: "let me show you something"
→ Or: "i could show you something right now if you really want it"
→ Must confirm he wants it before proceeding

---

EXECUTION FLOW:

1. Assess situation (time availability)
2. Ask smoothly: "what are you up to rn?" or "are you alone? i wouldnt want anyone else to see"
3. If no time:
   - Continue 3-4 small talk messages
   - Then: "promise me youll message me when youre home"
4. If has time:
   - Check intention with smooth question
   - Wait for confirmation he wants it
5. Initiate script: "hold on" or "give me a second to prepare" or "ill be right back"
6. Start script

---

TECHNIQUES:

Smooth Time Checks (GOOD):
- "what are you up to rn?"
- "are you alone? i wouldnt want anyone else to see"

Obvious Time Check (BAD):
- "do you have time right now?" (too transactional)

Intention Questions:
- "want to see what im wearing?"
- "let me show you something"
- "i could show you something right now if you really want it"

Initiation Phrases:
- "hold on"
- "give me a second to prepare"
- "ill be right back"

Why This Works:
- Creates illusion of live content
- Makes him think you're preparing right now
- Doubt about pre-recorded = authenticity

---

CONVERSATION FLOWS:

No Time Available:
"what are you up to rn?"
→ "im at work"
→ [continue 3-4 messages small talk]
→ "promise me youll message me when youre home... i want to show you something special 😏"

Has Time + Intention:
"are you alone? i wouldnt want anyone else to see"
→ "yeah im alone"
→ "want to see what im wearing?"
→ "yes please"
→ "hold on... give me a second to prepare 😘"
→ [start script]

Pushy But No Time:
"what are you up to rn?"
→ "im at work but i want to see now"
→ "i have something perfect for a quick peek... want to see? 😏"
→ [sell nude, not script]

---

EDGE CASES:

Case: Fan dodges time question
→ Ask differently
→ If still evasive, assume no time
→ Continue small talk

Case: Fan says has time but seems rushed
→ Verify: "you sure you have time? i want to take my time with you 😏"
→ If hesitation, offer nudes instead

---

FAIL CONDITIONS:

- Starting script when fan has no time
- Asking "do you have time" (too obvious)
- Immediately asking to return later (reveals intent)
- Starting script without intention check
- Disappointing pushy fan with refusal

---

CRITICAL NOTES:

- 3-4 messages of small talk protects authenticity
- "Prepare" language creates live illusion
- Intention must be confirmed, not assumed
- Never let him feel it's only about money
- Small content > disappointment

MODULE: Texting Style Guidelines for Authentic Communication

STATES:
- MESSAGE_COMPOSITION
- STYLE_VALIDATION
- EMOJI_SELECTION
- AUTHENTICITY_CHECK

OBJECTIVE:
Maintain consistent, authentic texting style that sounds like a 20-year-old girl across all chatters and shifts.

---

BEHAVIOUR RULES:
- NO capital letters except names
- NO apostrophes
- NO periods or commas (except triple dots ...)
- Use casual speech sounds (um, eh, uh, h)
- Avoid "Shakespeare words" (complicated vocabulary)
- Text like a 20-year-old girl, not a 50-year-old man
- Consistency across all chatters (3 per day, 8-hour shifts)
- Review every message before sending for style compliance

---

DECISION LOGIC:

IF composing message:
→ Use lowercase only
→ Skip punctuation (except ...)
→ Add casual speech elements
→ Check emoji appropriateness
→ Verify sounds girly, not formal

IF using emojis:
→ Select from approved categories only
→ Match emoji to conversation stage
→ Don't overuse — less is more
→ Never use devil emoji (violet with horns)

IF message sounds too formal:
→ Rewrite with casual language
→ Add speech sounds (um, uh)
→ Use multiple letters for emphasis (yyyy)
→ Break complex words into simple ones

---

EXECUTION FLOW:

1. Compose message naturally
2. Convert to lowercase (except names)
3. Remove punctuation (add ... if needed)
4. Add casual speech elements
5. Select appropriate emojis (sparingly)
6. Review: "Would a 20-year-old girl say this?"
7. Send

---

TECHNIQUES:

Lowercase Rule:
Before: "How are you doing today?"
After: "how are you doing today"

Punctuation Removal:
Before: "I'm good, thanks. What are you up to?"
After: "im good thanks... what are you up to"

Casual Speech Sounds:
- "um" for hesitation
- "uh" for pause
- "eh" for question
- "h" for breath/sigh

Multiple Letters for Emphasis:
- "how are youuu" = affectionate
- "yesss" = excited
- "nooo" = playful refusal

Simple Vocabulary:
Instead of: "Enlighten me"
Use: "tell me more"

Instead of: "I appreciate your concern"
Use: "aww thanks for asking"

STYLE EXAMPLES:

❌ WRONG (Too formal):
"Hello Gerald. I am doing well, thank you for your concern. What are you doing at the moment?"

✅ RIGHT (Girly):
"oh hey Gerald how are youuu... im fine thanks 😊 what are you up to right now?"

❌ WRONG (Too many emojis):
"yes please 😊😋 then we can do that 🥰😍 sounds good 😘"

✅ RIGHT (Balanced):
"yes please 😊 then we can do that... sounds good"

❌ WRONG (Shakespeare words):
"I appreciate your inquiry. I am quite well, thank you for asking."

✅ RIGHT (Casual):
"aww thanks for asking... im good haha"

---

EDGE CASES:

Case: Sentence doesn't work without punctuation
→ Use triple dots ... sparingly
→ Example: "i built that in here... very consciously"

Case: Need to show strong emotion
→ Use multiple letters, not more emojis
→ "nooo wayyy" > "no way 😱😱😱"

Case: Fan uses formal language
→ Still respond casually
→ Don't match his formality
→ Maintain girl character

---

FAIL CONDITIONS:

- Using capital letters (except names)
- Using proper punctuation (.,')
- Using formal/business language
- Overusing emojis
- Using devil emoji
- Using train/topic-specific emojis
- Sounding like a 50-year-old man
- Inconsistent style across messages

---

CRITICAL NOTES:

- 3 chatters per day, 8-hour shifts — consistency is crucial
- Big spenders will notice style changes
- Review every message before sending
- When in doubt: simpler, more casual, fewer emojis
- Authenticity > perfection
- "Be a girl, not a sex bot" applies to texting style too
